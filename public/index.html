About Project

Welcome to the Smart Parking System powered by Automatic License Plate Recognition (ALPR)! Our innovative solution leverages the capabilities of Google Cloud Vision API on a Raspberry Pi 4 to provide efficient and automated parking management.

How it Works:

  1. Vehicle Detection: An ultrasonic sensor constantly monitors the parking lot entrance, detecting arriving vehicles.
  2. Image Capture: Upon vehicle detection, a Raspberry Pi Camera Module captures a high-resolution image.
  3. License Plate Recognition: The image is processed using advanced image recognition algorithms, powered by Google Cloud Vision API, to accurately extract the license plate number.
  4. Data Logging and Display: The recognized license plate number, along with the timestamp, is securely logged. Our real-time dashboard provides an intuitive interface to monitor parking activity, including entry and exit times for each vehicle.

Key Features:

  • Automated License Plate Recognition
  • Real-time Parking Monitoring
  • Secure Data Logging
  • User-Friendly Dashboard
  • Scalable and Efficient
